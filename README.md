This project is a PLC-based 4-floor lift control system developed using SCL in TIA Portal. The system controls the complete operation of an elevator, including handling floor and cabin requests, determining the target floor, and managing the movement of the lift. When a user presses a button either from a floor or inside the cabin, the request is stored and processed by the PLC. Based on the current and target floor, the system decides the direction of movement (up or down) and controls the motor start, direction, and speed accordingly.

The lift uses floor sensors to detect its current position and stops automatically when it reaches the requested floor. Once the lift arrives, the motor stops and the door opens automatically. The door remains open for a defined time and then closes, while also allowing manual control through open and close buttons. A safety sensor ensures that if an object is detected during door closing, the door reopens immediately to prevent accidents. Additionally, the system includes safety interlocks to prevent movement when the door is open or when a drive fault occurs.

To enhance usability and monitoring, the system generates real-time status messages such as movement direction, current floor, and door status. It also includes a data logging feature that stores the last 10 system events using a circular buffer, ensuring that only new messages are recorded without duplication. The program is designed using a modular structure with User Defined Types (UDT) and Function Blocks (FB), making it scalable, reusable, and aligned with industrial automation standards.
